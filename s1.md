Guide de Mise en Forme du Texte avec CSSVoici les réponses à vos 20 questions sur la façon de styliser le texte en utilisant CSS.1. Comment changer la couleur du texte du premier paragraphe en rouge?Vous pouvez utiliser la propriété color pour définir la couleur du texte./* Cible le premier paragraphe (supposons qu'il a la classe .p1) */
.p1 {
  color: red;
}
2. Comment rendre le fond du premier paragraphe en bleu clair?Utilisez la propriété background-color pour changer la couleur d'arrière-plan..p1 {
  background-color: lightblue;
}
3. Comment aligner le texte du premier paragraphe au centre?La propriété text-align contrôle l'alignement horizontal du texte..p1 {
  text-align: center;
}
4. Comment appliquer une ombre au texte du second paragraphe?Utilisez text-shadow. La syntaxe de base est offset-x | offset-y | blur-radius | color./* Cible le second paragraphe (classe .p2) */
.p2 {
  /* Ajoute une ombre grise, 2px à droite, 2px vers le bas, avec 5px de flou */
  text-shadow: 2px 2px 5px grey;
}
5. Comment changer la police du texte du second paragraphe en "Roboto" (en utilisant Google Fonts)?Ceci se fait en deux étapes :Importer la police : Ajoutez un @import en haut de votre fichier CSS.Appliquer la police : Utilisez font-family pour l'appliquer à l'élément./* 1. À placer tout en haut de votre fichier CSS */
@import url('[https://fonts.googleapis.com/css2?family=Roboto&display=swap](https://fonts.googleapis.com/css2?family=Roboto&display=swap)');

/* 2. Appliquer la police au paragraphe */
.p2 {
  /* 'Roboto' est la police, 'sans-serif' est la police de repli */
  font-family: 'Roboto', sans-serif;
}
6. Comment augmenter l'espacement entre les lettres du second paragraphe?La propriété letter-spacing contrôle l'espace entre les caractères..p2 {
  letter-spacing: 3px; /* Ajoute 3 pixels d'espace entre les lettres */
}
7. Comment transformer le texte du troisième paragraphe pour que chaque mot commence par une lettre majuscule?Utilisez text-transform avec la valeur capitalize./* Cible le troisième paragraphe (classe .p3) */
.p3 {
  text-transform: capitalize;
}
8. Comment changer la direction du texte du troisième paragraphe pour qu'il soit affiché de droite à gauche?Utilisez la propriété direction..p3 {
  direction: rtl; /* "rtl" signifie "Right-To-Left" (droite à gauche) */
}
(Note : Pour des raisons sémantiques, il est souvent préférable de définir cela dans le HTML avec dir="rtl" sur l'élément, mais direction: rtl; est la réponse CSS.)9. Comment sous-ligner le texte du troisième paragraphe avec une ligne ondulée de couleur verte?Vous pouvez combiner les propriétés text-decoration pour obtenir cet effet..p3 {
  text-decoration-line: underline;  /* Type : souligné */
  text-decoration-style: wavy;     /* Style : ondulé */
  text-decoration-color: green;    /* Couleur : vert */
}
10. Comment augmenter la hauteur de ligne du texte du quatrième paragraphe?Utilisez line-height. Il est recommandé d'utiliser une valeur sans unité (multiplicateur) pour une meilleure accessibilité./* Cible le quatrième paragraphe (classe .p4) */
.p4 {
  /* 1.5 signifie 150% de la taille de la police */
  line-height: 1.5;
}
11. Comment indenter la première ligne du texte du quatrième paragraphe de 40 pixels?La propriété text-indent est faite pour cela..p4 {
  text-indent: 40px;
}
12. Comment rendre le texte du quatrième paragraphe en petites capitales?Utilisez la propriété font-variant..p4 {
  font-variant: small-caps;
}
13. Comment ajouter une ombre noire de 3 pixels à droite et 3 pixels vers le bas avec un flou de 5 pixels pour le texte du cinquième paragraphe?C'est une application spécifique de text-shadow./* Cible le cinquième paragraphe (classe .p5) */
.p5 {
  /* offset-x | offset-y | blur-radius | color */
  text-shadow: 3px 3px 5px black;
}
14. Comment rendre le texte du sixième paragraphe oblique?Utilisez font-style./* Cible le sixième paragraphe (classe .p6) */
.p6 {
  font-style: oblique;
}
15. Comment définir l'épaisseur du texte du sixième paragraphe à 600?La propriété font-weight contrôle la graisse du texte (400 = normal, 700 = gras)..p6 {
  font-weight: 600; /* 600 est "semi-bold" ou "demi-gras" */
}
16. Comment utiliser une couleur HSL pour le texte du sixième paragraphe avec une teinte de 120, une saturation de 100% et une luminosité de 50%?Utilisez la fonction hsl() pour la valeur de color..p6 {
  /* HSL = Teinte (Hue), Saturation, Luminosité (Lightness) */
  /* 120 est un vert vif */
  color: hsl(120, 100%, 50%);
}
17. Comment centrer verticalement le mot "essentiel" par rapport à la ligne de base du texte environnant?Pour cela, le mot "essentiel" doit être dans un élément <span> (ou un autre élément inline). Vous pouvez alors utiliser vertical-align./* Supposant un HTML comme :
   <p>Ceci est un mot <span class="mot-important">essentiel</span>.</p>
*/

.mot-important {
  vertical-align: middle;
}
18. Comment augmenter l'espacement entre les mots du texte du septième paragraphe?Utilisez la propriété word-spacing./* Cible le septième paragraphe (classe .p7) */
.p7 {
  word-spacing: 8px; /* Ajoute 8 pixels d'espace supplémentaire entre les mots */
}
19. Comment éviter les retours à la ligne dans le texte du septième paragraphe, même si cela signifie déborder de son conteneur?Utilisez white-space pour contrôler la façon dont les espaces blancs et les retours à la ligne sont gérés..p7 {
  white-space: nowrap; /* Empêche le texte d'aller à la ligne */
}
20. Comment ajouter une ombre double au texte du huitième paragraphe : une ombre noire décalée de 2 pixels vers le bas et une ombre rouge décalée de 4 pixels vers le haut?text-shadow accepte plusieurs ombres, séparées par des virgules./* Cible le huitième paragraphe (classe .p8) */
.p8 {
  text-shadow: 0px 2px black,  /* 1. Ombre noire : 0px h, 2px v */
               0px -4px red; /* 2. Ombre rouge : 0px h, -4px v (vers le haut) */
}
